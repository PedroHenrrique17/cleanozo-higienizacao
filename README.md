# cleanozo-higienizacao
meu projeto para empresa de higienização clean ozo

🧼 Cleanozo Higienização

Sistema/projeto desenvolvido para serviços de higienização, com foco em organização, praticidade e eficiência no gerenciamento ou execução dos processos.

🚀 Sobre o projeto

O Cleanozo Higienização tem como objetivo oferecer uma solução para auxiliar em atividades relacionadas à higienização, que impactam diretamente em uma melhora para nossa saúde.

- (Banco de dados:MySQL.)
